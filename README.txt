The 3D Glycano website can be found at: glycano.azurewebsites.net

The website works best on Google Chrome but also works on Opera.


Below are some examples of input that Glycano accepts:
==============================================
CASPER Format examples:


1. Linear: aDGlc(1->4)aDGlc


2. Branched: aDMan(1->4)[aDGal(1->6)]aDGlc


3. Repeating: ->6)aDMan(1->4)aDGlc(1->


4. Residue with substituents: aDGlc3Ac4OMe


5. Phosphate linkage: aDMan1P(O->6)aDMan


6. Linkage from the 2-postion: aDNeu5Ac(2->6)aDGlc

=================================================
